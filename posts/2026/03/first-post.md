# 첫 글

이 블로그는 마크다운으로 작성하고, 정적으로 빌드해서 GitHub Pages에 배포합니다.

## 코드 블록 예시

```ts
const title = "Hello static blog";
console.log(title);
```

앞으로 월별 폴더에 글을 추가하면 자동으로 아카이브 페이지에 반영됩니다.
